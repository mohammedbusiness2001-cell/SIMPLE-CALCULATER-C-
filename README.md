This project is a **learning-focused console application** built with C#.  
Its main goal is to practice **C# syntax** and apply **basic programming concepts** through a simple calculator.

##  Purpose
- Learn the basic syntax of C#
- Practice reading input from the console
- Apply conditional logic and basic operations
- Understand how simple programs are structured in C#

 ##  Features
- User enters the full mathematical expression (e.g. `10 + 5 * 2`)
- Supports basic operations:
  - Addition `+`
  - Subtraction `-`
  - Multiplication `*`
  - Division `/`
- Respects operator precedence
- Console-based application
